# Скрипты для работы с электронным дневником

Этот репозиторий содержит скрипты для учебного проекта электронного дневника школы.
Скрипты позволяют:

- Исправлять плохие оценки (двойки и тройки) на пятёрки.
- Удалять замечания.
- Добавлять похвалы от учителей.

Важно: проект учебный. Для работы нужен уже развернутый сайт электронного дневника и база данных.

---

## Использование.

### 1. Исправить оценки.

1. Скачайте `scripts.py` и положите рядом с `manage.py`.
2. Запустите Django shell:
3. Импортируйте скрипты:
```python
from scripts import fix_marks, remove_chastisements, create_commendation
from datacenter.models import Schoolkid
```
4.Найдите учетку:
```python
child = Schoolkid.objects.get(full_name__icontains='Фролов Иван')
```
5.Запустите функции:'
```python
fix_marks(child)
remove_chastisements(child)
create_commendation('Фролов Иван', 'Музыка')
```
6.Откройте сайт — оценки исправлены, замечания исчезли, появилась похвала.

## Помочь друзьям.

2. Помочь друзьям

Аналогично, указываем имя и предмет друга:
```python
create_commendation('Голубев Феофан', 'Математика')
```

3. Многократное использование

- Скрипты можно запускать сколько угодно раз.

- Плохие оценки исправляются повторно.

- Замечания удаляются снова.

- Похвала добавляется к последнему уроку по предмету.

4. Обработка ошибок ввода

Нет ученика:
```python
create_commendation('John', 'Музыка')
# => "Ученика с именем John не найдено"
```

Несколько учеников с одинаковым именем:
```python
create_commendation('Степан', 'Музыка')
# => "Найдено несколько учеников с именем Степан"
```

Неправильное название предмета:
```python
create_commendation('Фролов Иван', 'Матиматика')
# => "Уроки по предмету Матиматика для Фролов Иван не найдены"
```

5. Что за проект?
